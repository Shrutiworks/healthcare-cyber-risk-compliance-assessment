# Healthcare Cyber Risk & Compliance Assessment

A consulting-style Governance, Risk and Compliance (GRC) assessment conducted for a fictional healthcare organization (MedCare Hospital).

# Scope

- Critical Asset Identification  
- Threat Landscape Assessment  
- Vulnerability Assessment  
- Enterprise Risk Assessment Matrix  
- Security Control Recommendations  
- Compliance Framework Mapping  
- Residual Risk Assessment  

# Frameworks Used

- HIPAA  
- ISO 27001  
- PCI-DSS  
- GDPR  
- NIST Cybersecurity Framework  
- India DPDP Act 2023  

# Objective

To assess cybersecurity risks affecting healthcare infrastructure and recommend security controls aligned with regulatory compliance requirements.

# Project Type

Independent self-designed cybersecurity and compliance case study created for learning practical Governance, Risk and Compliance (GRC) concepts.
